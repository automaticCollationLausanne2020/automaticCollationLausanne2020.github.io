Introduction to automatic collation

This is the home page of the online workshop Introduction to automatic collation that will take place the 24th and 25th of september. This course is offered by the Programme Doctoral en Études Numériques of the University of Lausanne. From this home page you can access:

  • The program of the workshop with links to the slides for each session.
  • The Github repository that contains all the materials of the workshop.
  • The installation instructions of the software we will use (not required to follow the workshop)
  • Additional information about the tecnhological framework we will use during the course

Please forward any doubts, problems or suggestions to the course instructors: Elena Spadini (UNIL) - Elisa Nury (UNIGE/UNIL) - Helena Bermúdez Sabel (UNIL)
